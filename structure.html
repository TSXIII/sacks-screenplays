The Basics of Plot Structure

Before you can write your first screenplay, you must understand the basics of plot structure. Traditionally, films' stories are told in three acts:

  • Act I

    1. The main character is introduced.
    2. The main character's world is established.

  • Act II

    1. The main character's circumstances change.
    2. The main character is dealt a tough blow.

  • Act III

    1. The main character's journey concludes.
    2. All plot points are resolved.

These three acts can be further broken down; there are several additional components to each one:

Act I
Element Description
Setup The main character is introduced, and we explore his/her world.
Inciting Incident The main character experiences something that changes his/her circumstances.
Debate The main character decides whether or not to participate in the next leg of his/her journey.
Turning Point The main character experiences something that forces him/her to leave the familiar behind.
Act II
Element Description
Subplot The main character has another element added to his/her journey, such as romance.
Midpoint The main character's situation grows more dire.
All is Lost The main character experiences something that brings him/her to his/her lowest low.
New Plan The main character discovers another path to victory.
Act III
Element Description
Climax The main character faces his or her biggest obstacle.
Denouement The main character settles into his/her new way of life, post-climax.

All films are different, but most of them use this model as a foundation for their plots. Try implementing it when planning your story! It is an excellent tool for screenwriters of every level.
